ReadMe für backup.sh
---------------------

INSTALLATION
1.) Script hochladen und ggf. die Angaben im Script an die Bedürfnisse anpassen
2.) Script aufrufen, evtl. mittels Cronjob regelmäßig starten lassen

Beispiel für einen CronJob:
Die folgende Zeile startet das Backup-Script jeden Morgen um 3.00 Uhr:

00 3 *** root /root/bin/backup.sh


Viel Spaß bei der Benutzung.
